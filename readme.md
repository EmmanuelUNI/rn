# Trabajo RN

## Requisitos
- Node.js 18+
- Expo CLI (`npm install -g expo-cli`)
- App **Expo Go** en el celular

## Instalación

```bash
npm install
```

## Correr

```bash
npx expo start
```

Escanea el QR con Expo Go (Android) o la cámara (iOS).


## Variables de entorno

Crea un archivo `.env` en la raíz:

```
EXPO_PUBLIC_ROBLE_PROJECT_ID=tu_project_id
```
